# <a name="recommended-adapters"></a>Adaptadores recomendados
> [!NOTE]
> Se va a actualizar, con lo que se realizarán adiciones a y es posible que se revisen estas recomendaciones del adaptador de equipo de Windows Mixed Reality. Vuelva a comprobar periódicamente la versión más reciente.

Es posible que el [equipo compatible con Windows Mixed Reality](windows-mixed-reality-minimum-pc-hardware-compatibility-guidelines.md) no tenga los puertos HDMI y USB 3,0 de tamaño completo necesarios para conectar los auriculares que se incluyen. En ese caso, necesitará adaptadores para conectar los auriculares. Es posible que algunos adaptadores también requieran la adición de un par para conectarse a su PC o al cable del casco. A continuación encontrará una lista de tipos de adaptador que puede necesitar y algunas recomendaciones sobre modelos de adaptador específicos.

## <a name="windows-mixed-reality-compatible-usb-bluetooth-adapters"></a>Adaptadores de Bluetooth USB compatibles con Windows Mixed Reality

Si el casco se lanzó después de 2018, los adaptadores no son necesarios y los controladores se emparejan con los auriculares.

Los dispositivos enviados en 2017 o una versión anterior necesitan un adaptador para los controladores. Si el equipo no incluye Bluetooth para emparejar [controladores de movimiento de la realidad mixta de Windows](controllers-in-wmr.md), puede usar un adaptador como el adaptador USB Bluetooth 4,0 Low Energy micro.

> [!NOTE]
> Microsoft no fabrica, prueba, certifica ni garantiza la funcionalidad, la confiabilidad o la calidad de ningún adaptador. Si tiene preguntas, problemas o problemas con cualquier adaptador, debe ponerse en contacto con el vendedor o el fabricante del adaptador.

## <a name="using-external-usb-30-hubs-with-windows-mixed-reality-headsets"></a>Uso de los concentradores USB 3,0 externos con auriculares de realidad mixta de Windows

Los auriculares Windows Mixed Reality se pueden usar con un concentrador USB 3,0 externo si el equipo no tiene suficientes puertos USB abiertos. El concentrador USB 3,0 externo **debe alimentarse externamente a través de un adaptador de AC** para proporcionar suficiente energía al casco.

Además, tenga en cuenta la siguiente guía para el uso de un concentrador USB 3,0 externo: 
* El concentrador USB debe ser un concentrador USB 3,0.
* El concentrador USB 3,0 debe estar conectado a un puerto USB 3,0 del equipo.  
* El concentrador USB 3,0 debe estar alimentado externamente (tenga un adaptador de AC que proporcione alimentación adicional).
* Se sabe que el concentrador USB 3,0 SuperSpeed de 10 puertos que se pueda conectar con el adaptador de alimentación de 48W funciona bien con auriculares de realidad mixta de Windows y la mayoría de los equipos.
