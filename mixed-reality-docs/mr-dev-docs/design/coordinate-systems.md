## <a name="spatial-coordinate-systems"></a><span data-ttu-id="34586-191">Sistemas de coordenadas espaciales</span><span class="sxs-lookup"><span data-stu-id="34586-191">Spatial coordinate systems</span></span>

<span data-ttu-id="34586-192">Todas las aplicaciones de gráficos 3D usan [sistemas de coordenadas cartesianos](https://docs.microsoft.com/windows/uwp/graphics-concepts/coordinate-systems) para motivar las posiciones y las orientaciones de los objetos virtuales.</span><span class="sxs-lookup"><span data-stu-id="34586-192">All 3D graphics applications use [Cartesian coordinate systems](https://docs.microsoft.com/windows/uwp/graphics-concepts/coordinate-systems) to reason about the positions and orientations of virtual objects.</span></span> <span data-ttu-id="34586-193">Estos sistemas de coordenadas establecen tres ejes perpendiculares para colocar los objetos: un eje X, Y y Z.</span><span class="sxs-lookup"><span data-stu-id="34586-193">These coordinate systems establish 3 perpendicular axes along which to position objects: an X, Y, and Z axis.</span></span>

<span data-ttu-id="34586-194">En la [realidad mixta](../discover/mixed-reality.md), los motivos de las aplicaciones son los sistemas de coordenadas virtuales y físicos.</span><span class="sxs-lookup"><span data-stu-id="34586-194">In [mixed reality](../discover/mixed-reality.md), your apps reason about virtual and physical coordinate systems.</span></span> <span data-ttu-id="34586-195">Windows llama a un sistema de coordenadas que tiene un significado real en el mundo físico, un **sistema de coordenadas espaciales**.</span><span class="sxs-lookup"><span data-stu-id="34586-195">Windows calls a coordinate system that has real meaning in the physical world a **spatial coordinate system**.</span></span>
